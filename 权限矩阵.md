> 🔄 2026-06-12 调整：**销售部 / 电工部 / 装配组** 只保留「本部门菜单 + 项目目录」，**不再有项目详单菜单**；且其在**项目目录中的项目编号仅展示、不可点进项目详单**（其余有详单权限的角色可点）。设计师、生产主管仍可进详单。

## 三、关键功能权限

| 功能 | 可操作角色 |
|---|---|
| 销售下单（建项目+台账+派部门） | 销售员 / 销售主管 / 管理层 |
| 销售台账**编辑** | 销售主管 / 管理层 |
| 销售台账**合计** + 按销售员筛选 | 销售主管 / 管理层 |
| 上传合同 / 开票申请 | 销售员 / 销售主管 / 管理层（看自己范围） |
| 项目目录「＋下单」浮动按钮 | **仅管理层**（派设计/电工）；生产部改由销售派单，不在项目目录下单 |
| **项目目录「项目编号」→ 进入项目详单** 🔄 | 有项目详单权限者可点（管理层/设计/生产主管/钣金/采购/仓库/物流/财务）；**销售 / 电工 / 装配组 / 售后 仅展示不可点**（openDetail 闸门 + 全览弹窗「进入详情」按钮亦隐藏） |
| 部门负责人**分派** | 设计部负责人（设计）· 电工部负责人（电工）· **生产部主管（生产）** ← 三部门一致 |
| **部门报表**（仅本部门数据，页头醒目主色按钮） | 设计/电工/生产 部门负责人各自 + 管理层 |
| **销售报表**（全部销售项目：合同额/开票进度/收款计划/按销售员业绩/客户分类，页头醒目主色按钮） | 销售主管 / 管理层 |
| **接单回传**（员工接单后姓名写回项目目录：设计→设计师列、电工→电工列） | 设计师 / 电工（系统自动） |
| **月度工作报表**（全公司） | **仅管理层** |
| 仓库出入库 / 物料 / 收发存 / 发货清单 | 仓库员 / 仓库主管（写）· 管理层（读）· 设计师（只读查库存） |
| 设计完成「四表校验」+ 上传产物 | 设计师（本人任务） |
| 物流确认发货（回写发货日期） | 物流发货部 |
| **售后登记**（按项目登记售后问题 + 费用） 🆕 | 售后部员工 / 管理层 |
| **售后费用审批**（通过→自动同步财务部 + 推送） 🆕 | 售后部主管 / 管理层 |
| **售后费用查看**（按项目展示售后问题 + 费用） 🆕 | 财务部（已审批同步）/ 管理层 |
| **电工采购单（项目详单第5张表，固定列名）** 🆕 | 电工部上传生成；采购/仓库补充采购负责人·到货·签字；详单可见者只读 |
| **售后物料清单上传 → 回传财务部** 🆕 | 售后部员工登记时必传；审批后随费用/问题同步财务部「售后费用」卡片（可下载） |
| **装配前置表完成统计**（钣金装配/标准件清单/外协外购 三表完成度→装配组工作台） 🆕 | 来源=项目详单各表；管理层/生产主管/设计师可在详单「标记已完成/进行中」；装配组只读查看 |
| **进行中任务「换人」**（防离职/请假，转交同部门他人，回传项目目录） 🆕 | 各部门负责人（设计/电工/生产主管）+ 管理层，在「任务跟踪」表操作 |
| 导出（项目/报表/库存等） | 管理层**免审**；其余角色→申请→管理层审批 |
| 完成可逆 / 冲红 / 列隐藏(制图) | 见各模块（制图列对业务无感知=逻辑删除） |
| 消息推送人（接收逾期/预警推送） | 销售主管 / 各部门主管 / 仓库主管（在权限管理标记） |
