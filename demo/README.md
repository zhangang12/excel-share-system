# 项目管理系统 · 单机演示版

> 用于客户现场演示，全部数据存本地 SQLite，无需服务器、无需联网（首次安装除外）。

## 演示前的一次性准备

**你需要在演示笔记本上安装**：

| 软件 | 版本 | 下载 |
|---|---|---|
| Python | 3.10+ | https://www.python.org/downloads/ （安装时**勾选 Add Python to PATH**）|
| Node.js | 20 LTS | https://nodejs.org/zh-cn/download |

### 步骤

1. 把整个项目文件夹（含 `backend/` `frontend/` `demo/`）放到笔记本任意位置
2. 进入 `demo/` 目录
3. **双击 `setup.bat`**（首次约 3-5 分钟，自动装依赖 + 构建前端）
4. 看到 "✅ 安装完成" 即可

## 日常启动（每次演示）

进入 `demo/` 目录，**双击 `start.bat`**。

启动后：
- 浏览器自动打开 http://127.0.0.1:8000
- 用 `admin / admin123` 登录
- 首次登录会要求改密（演示时可继续用 admin123，跳过此步：直接关掉对话框）

## 演示流程建议

| 阶段 | 操作 | 演示要点 |
|---|---|---|
| 1. 登录 | admin / admin123 | 角色 + 部门体系 |
| 2. 项目一览 → 导入汇总表 | 上传 `2026项目倒计时.xlsx` | 自动建项目 + 识别字段类型 + 分"进行中/已完成" |
| 3. 点项目编号跳转 | 点击 2026-039 | 跨页跳转 |
| 4. 项目详情 → 上传模版 | 上传 `项目编号：2026-039.xls` | 自动建 3 个 sub-sheet |
| 5. 网格编辑 | 改单元格、加行、加列 | 像 Airtable / 飞书多维表格 |
| 6. 字段权限 | admin 在列头 🔒 → 给"采购员"列只对采购部可见 | 字段级权限演示 |
| 7. 实时协作 | 两个浏览器窗口同登录 | 一个改另一个秒看到 |
| 8. 导出 | 导出 .xlsx | 数据完整还原 |
| 9. 操作审计 | admin → 操作审计 | 所有操作可追溯 |

## 重置演示环境

如果客户演示完想还原干净状态：

```
关闭 start.bat 窗口
删除 ..\backend\data\app.db
再启动 start.bat → 会自动重新初始化（admin/admin123 + 内置部门）
```

## 端口冲突

如果 8000 端口被占用，编辑 `start.bat` 把 `--port 8000` 改成别的（如 `--port 18000`），同时修改浏览器自动打开的 URL。

## 常见问题

**Q：双击 setup.bat 闪退**
A：可能是 Python/Node 未安装。打开 cmd 跑 `python --version` 和 `node --version` 验证。

**Q：start.bat 报"端口已被占用"**
A：换一个端口（见上）

**Q：演示中数据丢了**
A：SQLite 文件在 `backend/data/app.db`，可备份

**Q：能离线演示吗**
A：可以。第一次 setup 需要联网装依赖，之后纯离线。
