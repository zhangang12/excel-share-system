一、模块目标与范围

把现有「采购部」从"看图纸/数据表"升级为完整的采购管理子系统：管供应商、走采购全流程（请购→下单→到货→对账→付款）、出每家供应商的对账报表、并与财务部应付/付款联动，最终沉淀月/季/年采购额汇总报表。

范围内

供应商档案、采购申请/订单、到货登记、供应商对账单、应付与付款（财务联动）、采购汇总报表。

暂不含（可二期）

在线询价比价竞标、供应商门户自助、复杂质检/退换货、与外部税务系统对接。

二、角色与权限

#角色主要权限业务确认
1采购员（外协/标准件/原料/激光等分工）维护供应商、建采购申请/订单、登记到货、生成对账单；只看本人/本类数据（沿用现有按账号分工）
2采购主管审批采购订单、审核对账单、看全量采购报表
3财务应付账款、发票校验、付款审批/登记；接收对账推送
4管理层全量可见、汇总报表、审批兜底

三、采购全流程

① 请购
项目/部门提需求→ ② 选供应商
报价/比价(可选)→ ③ 采购订单
主管审批→ ④ 下单给供应商→ ⑤ 到货登记
数量/质检→ ⑥ 月度对账
供应商对账单→ ⑦ 财务核账/发票→ ⑧ 付款 完成

说明：小额/紧急采购可走简化流（采购员直接建单→主管事后审）。请购可由现有项目流程自动带出（如设计推过来的外协/原料需求）。

四、功能模块 & 需求清单

4.1 供应商管理（对应需求③）

#功能点说明优先级业务确认
1新增/编辑供应商编号、名称、分类（外协/标准件/不锈钢原料/激光件/电气件…）、联系人、电话、地址、税号、开户行+银行账号、付款账期、状态（启用/停用）高
2供应商资质附件营业执照、合同、资质证书上传留存中
3供应商分类/标签按品类筛选；采购员按分工只看相关供应商高
4供应商评级/绩效到货及时率、质量、价格，给星级（供选型参考）低
5停用而非删除有历史采购的供应商不可物理删除，只能停用（保数据可追溯）中

4.2 采购申请 / 采购订单

#功能点说明优先级业务确认
6新建采购订单选供应商+关联项目；明细行：物料名称/规格/数量/单位/单价/金额；自动算总额、税额高
7从项目需求带出请购设计/电工推过来的外协/原料/激光件需求 → 一键转采购单（少录一遍）中
8采购订单审批超额/按规则需采购主管审批；小额可免审（可配置阈值）高
9订单状态流转待审批 → 已下单 → 部分到货 → 已到货 → 已对账 → 已付款高
10打印/导出采购单（PDF）带公司抬头，发给供应商中
11比价（可选）同一需求录多家报价，择优下单（二期）低

4.3 到货 / 收货

#功能点说明优先级业务确认
12到货登记按采购单登记到货数量、到货日期；支持分批到货高
13与仓库联动到货→仓库入库（沿用现有仓库模块），避免重复登记中
14到货异常/质检登记不合格/退货，影响供应商评级（二期）低

4.4 供应商对账 & 报表（对应需求①，重点）

#功能点说明优先级业务确认
15每家供应商独立对账单按期间（月/自定义）汇总该供应商所有采购明细：采购笔数、采购总额、已付、应付余额高
16对账单明细可下钻点开看每张采购单/每次到货明细，可导出 Excel/PDF 发供应商核对高
17对账确认采购/供应商双方核对无误后"确认对账"，锁定本期，生成应付高
18供应商往来余额累计应付/已付/余额一目了然高

4.5 财务联动（对应需求②）

#功能点说明优先级业务确认
19对账确认 → 推送财务应付本期对账确认后自动生成"应付账款"并推送/通知财务（站内+企微）高
20发票校验供应商开来的发票录入/上传，与对账金额勾稽（金额、税率）中
21付款申请 → 财务审批 → 付款登记采购发起付款申请，财务审批并登记付款（部分付款/全额）；回写采购单与对账余额高
22应付账款汇总进财务各供应商应付汇总纳入财务报表口径（与销售开票同一财务视图）高

4.6 采购汇总报表（对应需求④，重点）

#功能点说明优先级业务确认
23采购额汇总（月/季/年）按时间维度统计采购总额、到货额、已付额、应付额；趋势图高
24按供应商排名Top 供应商采购额、占比，识别核心供应商高
25按品类/项目分布外协/标准件/原料/激光件各占多少；按项目看采购成本中
26应付账龄应付按账龄分段（30/60/90天+），提示逾期付款风险中
27报表导出所有报表可导出 Excel/PDF中

六、与现有系统衔接 & 分期建议

衔接点

  • 现有「采购部」按账号分工（外协/标准件/激光等）→ 直接复用为采购员分工 + 供应商分类。
  • 设计/电工推来的外协加工/不锈钢原料/激光件需求 → 转为请购/采购单来源。
  • 到货 → 复用现有「仓库」入库；财务联动 → 复用现有财务/开票的同一套财务口径。
  • 所有通知走现有「站内消息 + 企业微信」推送通道。

分期

期内容业务确认
一期(核心)供应商管理 + 采购订单 + 到货登记 + 供应商对账单 + 月/季/年采购汇总 + 财务应付推送
二期付款审批闭环 + 发票校验 + 账龄报表 + 供应商评级
三期比价竞标 + 质检退换 + 供应商门户
